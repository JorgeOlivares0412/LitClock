# LitClock

The time, told through literature.

A static web app built with SvelteKit, deployed to GitHub Pages. Every minute, a new quote from a novel that mentions that exact time.

## Setup

```bash
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions handles the rest.

> **Note:** Update the `base` path in `svelte.config.js` to match your GitHub repo name.
